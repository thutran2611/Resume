Bash and Git

June 08, 2019

Most people are used to graphical user interface since it's visual. You can see what you change in your directory. However, in order to work as a developer, being able to master command line interface (CLI) is a must. Shell is a category of programming languages designed for CLI. Bash is the most popular shell. Bash helps developers keep track of and manipulate their directories and files a lot faster. However, it can be challenging with the command lines.

Git is a the most popular version control system. A repository is a directory that is "git enabled". A commit is a "frozen instance" in time. Hash is a number and letter combo that uniquely IDs a commit.
